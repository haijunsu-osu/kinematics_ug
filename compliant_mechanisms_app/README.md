# ME 7751 Compliant Mechanisms Course App

A modern web application for exploring ME 7751 Compliant Mechanisms course materials, including lecture slides, videos, and interactive tools.

## Course Modules

### Module 1: PRB Models
**Pseudo Rigid Body Models for planar compliant mechanisms**

This module covers the fundamentals of pseudo-rigid-body models (PRBM), which simplify the analysis of compliant mechanisms by approximating flexible members as rigid links with lumped compliance at discrete joints.

### Module 2: Screw Theory
**Screw theory approach for design and analysis of spatial compliant mechanisms**

This module covers advanced methods using screw theory for analyzing and synthesizing spatial compliant mechanisms and flexure systems.

## Development

### Prerequisites
- Node.js 18+
- npm

### Installation
```bash
npm install
```

### Running the Development Server
```bash
npm run dev
```

### Building for Production
```bash
npm run build
```

The built files will be in the `dist` directory.
