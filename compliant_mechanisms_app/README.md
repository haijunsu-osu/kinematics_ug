# Compliant Mechanisms Course App

A modern web application for exploring compliant mechanisms course materials, including lecture slides, videos, and interactive tools.

## Course Modules

### Module 1: PRB Models
**Pseudo Rigid Body Models for planar compliant mechanisms**

This module covers:
- Introduction to Compliant Mechanisms
- Review of Mechanics of Materials for Compliant Mechanisms
- Pseudo Rigid Body Models
- Kinetostatic Analysis

### Module 2: Screw Theory
**Screw theory approach for design and analysis of spatial compliant mechanisms**

This module covers:
- Screw Theory Review
- Constraint and Screw Theory for Flexures
- Synthesis of Flexure Mechanisms using Screw Theory
- Compliance Matrices for Flexure Design

## Development

### Prerequisites
- Node.js 18+
- npm

### Installation
```bash
npm install
```

### Running the Development Server
```bash
npm run dev
```

### Building for Production
```bash
npm run build
```

The built files will be in the `dist` directory.
